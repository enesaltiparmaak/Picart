AI destekli, full-stack resim düzenleme uygulaması.

[https://picart-sixfinger.vercel.app](https://picart-sixfinger.vercel.app)

- Katmanlı düzenleme

## Kurulum
```bash
cd frontend
yarn install
yarn start
```
